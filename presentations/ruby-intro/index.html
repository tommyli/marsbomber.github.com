Ruby Syntax

Variables have no leading $

fruit = "banana"

Statements have no tailing ;

fruit = "banana"

Parentheses are optional

s = String.new "Banana"
s.concat " Boat"

Punctuation has meanings

> 1.odd?        #=> true
> a = [1, 2]    #=> [1, 2]
> a.reverse     #=> [2, 1]
> a             #=> [1, 2]
> a.reverse!    #=> [2, 1]
> a             #=> [2, 1]

Object Oriented

Everything is an object. That includes booleans, number and even nil (null in php). This means more concise syntax. Always use method calls on objects, no function calls.

> true.class  #=> TrueClass
> 7.class     #=> Fixnum
> nil.class   #=> NilClass

This allows us to do some cool things.

> 2.times {|n| puts "#{n}: Ruby is cool"}
0: Ruby is cool
1: Ruby is cool

Open Class

Ruby allows you to open any predefined classes to overwrite or amend methods.

Powerful but DANGEROUS

  • Use in framework/library code. e.g. Dynamically define methods
  • Handle with care in application code! Normally there are explicit but clearer ways to achieve the same results. (Yes, that reads application DESIGN)

Sample

> class Test; def say_one; puts "one"; end; end
> a = Test.new           #=>
> a.say_one              #=> one
> a.say_two              #=> NoMethodError: undefined method `say_two'
> class Test; def say_two; puts "two"; end; end
> a.say_two              #=> two

Goodies I love

String

> "%s %s!" % %w[String interpolation]    #=> "String interpolation!"
> %q[auto escape "]                      #=> "auto escape \""
> "   space   around   ".squeeze         #=> " space around "
> a.say_two                              #=> two
> "a b 1 2".gsub(/[a-z]/) { $&.upcase }  #=> "A B 1 2"

Array

> a = [1, 2, 3, 4]             #=> [1, 2, 3, 4]
> a.select {|e| e > 2}         #=> [3, 4]
> a.select(&:odd?)             #=> [1, 3]
> a.reject(&:odd?)             #=> [2, 4]
> a.inject(&:+)                #=> 10
> b = [nil, '', nil, 1, 3, 1]  #=> [nil, "", nil, 1, 3, 1]
> b.compact                    #=> ["", 1, 3, 1]
> b.uniq                       #=> [nil, "", 1, 3]
> a & b                        #=> [1, 3]
